# 4. 思维导图 & 流程图

# 方法一：

  1. 使用Markmapopen in new window生成思维导图html文件
  2. 将html文件放在docs/.vuepress/public/markmap/
  3. 通过<iframe>插入到Markdown
  <iframe :src="$withBase('/markmap/01.html')" width="100%" height="400" frameborder="0" scrolling="No" leftmargin="0" topmargin="0"></iframe>
1

# 方法二：

通过<iframe>标签引入processonopen in new window或其他在线作图工具生成的链接。

<iframe src="https://www.processon.com/view/link/5e718942e4b015182028682c" width="100%" height="500" frameborder="0" scrolling="No" leftmargin="0" topmargin="0"></iframe>
1

# 方法三：

使用流程图插件：

  • vuepress-plugin-flowchartopen in new window
  • vuepress-plugin-mermaidjsopen in new window

# 5.Demo演示框

# 方法一：

  1. 安装 vuepress-plugin-demo-blockopen in new window或其他同类插件，使用方法看插件文档

  3. Markdown中使用

同类插件：vuepress-plugin-demo-containeropen in new window
