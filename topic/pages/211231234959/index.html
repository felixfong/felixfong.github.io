# 0. 前言

TIP

net：网络数据的基本操作。 Http包整体主线：提供可扩展的HTTP服务器和基础客户端，解析HTTP请求和回复。即接收客户端请求，并向客户端返回响应。 runtime：Go程序运行时的交互操作，例如垃圾回收和协程创建。 reflect：实现通过程序运行时反射，让程序操作任意类型的变量。
